# Simple ATM System

A Java-based console application for managing basic bank account transactions.

## 🚀 Features
* **Check Balance:** View current account funds instantly.
* **Deposit:** Safely add money to the account.
* **Withdraw:** Remove funds with automated balance verification.
* **Robust Input:** Handles invalid characters and negative numbers.

## 📂 Project Structure
* `BankAC.java`: Manages the account balance and data.
* `ATM.java`: Controls the logic for banking operations.
* `FinalELC.java`: The main driver class that runs the user interface.

## 🛠️ Installation & Running
1. **Compile the program:**
   javac FinalELC.java
2. **Run the program:**
   java FinalELC
