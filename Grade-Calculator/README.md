# Student Grade Calculator

A Java console application that calculates student performance based on marks obtained in various subjects.

## 🚀 Features
* **Dynamic Input:** Handles any number of subjects defined by the user.
* **Smart Validation:** Ensures marks entered are within the valid range (0-100).
* **Automated Calculations:** Computes total marks and average percentage automatically.
* **Grade Assignment:** Assigns letter grades (A+ to F) based on standardized percentage brackets.
* **Robust Input:** Prevents program crashes from non-numeric user input.

## 📂 Project Structure
* `GradeCalculate.java`: The logic class that handles mathematical operations and grade determination.
* `GradeShowELC.java`: The main entry point (Execution Logic Class) that manages user interaction and reports.
